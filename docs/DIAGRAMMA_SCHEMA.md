## рҹ”Қ **DOVE SONO I DATI?**

### **Nel DATABASE:**

```sql
-- Quando intestatario = 'richiedente':
SELECT 
  nomeRichiedente,           -- Nome
  cognomeRichiedente,        -- Cognome
  cfIntestatario,            -- CF
  indirizzoIntestatario,     -- Indirizzo
  cittaIntestatario,         -- CittГ 
  capIntestatario,           -- CAP
  provinciaIntestatario      -- Provincia вӯҗ
FROM leads 
WHERE id = 'LEAD-XXX';

SELECT 
  nomeAssistito,             -- Nome

  cittaAssistito,            -- CittГ

  provinciaAssistito         -- Provincia вӯҗ
FROM leads 
WHERE id = 'LEAD-XXX';
```
